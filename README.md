# ARTI 308 – Lab 2
Identifying ML Problems, Selecting Open Datasets, and Methodology Design

## Dataset
Students Performance Dataset from Kaggle.

## Machine Learning Problem
This project focuses on identifying a machine learning problem using an open dataset.
The selected problem is a regression task aiming to predict students’ academic performance
based on demographic, educational, and behavioral features.

## Tasks Completed
- Selected an open dataset from Kaggle
- Identified the machine learning problem type
- Loaded and inspected the dataset using Pandas
- Designed a methodology diagram
- Created a Kaggle Notebook for data inspection
